# Panduan Kontribusi

Terima kasih telah tertarik untuk berkontribusi pada proyek **Minecraft Fabric Server 1.21.1**! Kami sangat menghargai bantuan Anda untuk membuat panduan ini menjadi lebih baik dan bermanfaat bagi komunitas.

## Bagaimana Cara Berkontribusi?

### 1. Melaporkan Bug
Jika Anda menemukan kesalahan dalam panduan atau masalah teknis pada konfigurasi server:
- Periksa halaman [Issues](https://github.com/Nugraa21/Server-Minecraft-Fabric-1.21.1/issues) untuk melihat apakah masalah tersebut sudah dilaporkan.
- Jika belum, buat Issue baru dengan deskripsi yang jelas, langkah-langkah untuk mereproduksi masalah, dan hasil yang diharapkan.

### 2. Mengusulkan Fitur atau Perbaikan
Kami menerima saran untuk penambahan tutorial, mod baru yang bermanfaat, atau perbaikan struktur file:
- Fork repository ini.
- Buat branch baru untuk fitur Anda (`git checkout -b fitur/FiturBaru`).
- Lakukan commit pada perubahan Anda (`git commit -m 'Menambahkan fitur XYZ'`).
- Push ke branch tersebut (`git push origin fitur/FiturBaru`).
- Buat Pull Request.

### 3. Memperbaiki Dokumentasi
Jika Anda menemukan kesalahan pengetikan atau ingin memperjelas bahasa dalam README:
- Anda dapat langsung menyarankan perubahan melalui Pull Request.

## Standar Kode dan Etika
- Gunakan bahasa yang sopan dan profesional dalam setiap interaksi.
- Pastikan setiap mod yang diusulkan kompatibel dengan versi Fabric 1.21.1.
- Hindari menyebarkan tautan unduhan yang tidak resmi atau berbahaya.

Terima kasih telah membantu proyek ini berkembang!
