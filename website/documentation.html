Penjelasan mendalam mengenai infrastruktur, batasan, dan solusi umum dalam penggunaan server ini.

Mengapa GitHub Codespaces?

Server ini dirancang untuk berjalan di atas infrastruktur GitHub Codespaces. Ini adalah lingkungan pengembangan berbasis cloud yang disediakan oleh GitHub (Microsoft).

Keuntungan Utama:
  • Gratis (Tier Free): Setiap pengguna GitHub mendapatkan kuota penggunaan bulanan gratis (sekitar 120 jam core untuk free tier).
  • Spesifikasi Tinggi: Standar mesin yang diberikan memiliki RAM dan CPU yang sangat mumpuni (biasanya 2-core atau 4-core dengan RAM 4GB-8GB) untuk menjalankan server Minecraft.
  • Koneksi Cepat: Terhubung langsung dengan infrastruktur Azure, memberikan kecepatan download/upload yang tinggi.

Dengan menggunakan Codespaces, kita menghilangkan kebutuhan untuk menyewa VPS mahal atau membebani komputer pribadi Anda (localhost) yang harus menyala terus menerus.

Batasan Kuota & Penyimpanan

Meskipun sangat powerful, GitHub memberikan batasan pada akun gratis (Free Tier). Sangat penting untuk memahami ini agar server Anda tidak tiba-tiba berhenti.

Storage (Penyimpanan)

Github memberikan batas penyimpanan sekitar 15 GB per bulan untuk Codespaces.

Implikasi: Jika ukuran world Minecraft Anda menjadi sangat besar (bergigabyte), atau Anda menginstal terlalu banyak mod berat, kuota storage ini bisa cepat habis. Selalu pantau penggunaan disk Anda.

Core Hours (Jam Pakai)

Batas penggunaan komputasi sekitar 120 jam core per bulan.

Implikasi: Server tidak bisa menyala 24/7 selamanya secara gratis. Jika Anda menjalankannya terus menerus, kuota gratis akan habis dalam beberapa hari. Matikan server (Stop Codespace) saat tidak dimainkan.

Peringatan Hibernasi: Codespaces akan otomatis "mati" (hibernate) jika tidak ada aktivitas (seperti mengetik di terminal) setelah 30 menit secara default. Anda perlu menggunakan extension atau script keep-alive jika ingin meninggalkannya sebentar, namun tetap disarankan untuk mematikannya jika tidak dipakai.

Masalah Umum & Solusi

Server Lag / TPS Drop

Biasanya terjadi saat generate chunk baru. Pastikan Anda sudah menginstal mod Lithium dan FerriteCore. Jika masih berat, kurangi view-distance di file server.properties menjadi 8 atau 10.

IP Berubah-ubah

Setiap kali Anda me-restart tunnel playit.gg, alamat IP/domain mungkin tetap sama jika Anda sudah claim agent, tapi jika membuat tunnel baru, IP akan berubah. Pastikan agent playit berjalan di background.

Java Version Error

Minecraft 1.21.1 WAJIB menggunakan Java 21. Jika Anda melihat error versi mayor/minor class file, itu berarti Anda mencoba menjalankan dengan Java 17 atau 8. Cek dengan java -version.

"Failed to bind to port"

Artinya ada server lain yang sedang berjalan di port yang sama (25565). Matikan proses java yang lain atau restart codespace Anda dengan sudo reboot.
