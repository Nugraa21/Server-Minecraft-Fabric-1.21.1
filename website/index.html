Instalasi Langkah-demi-Langkah

1

Persiapan Awal

Pastikan Anda telah melakukan fork repository ini ke akun GitHub Anda sendiri sebelum memulai di Codespaces.

Penting: Jangan skip langkah ini agar progress tersimpan di akun Anda.
2

Update Sistem & Install Java 21

Minecraft 1.21.1 memerlukan Java 21. Jalankan perintah ini di terminal:

bash
sudo apt update && sudo apt upgrade -y
sudo apt install openjdk-21-jdk -y
java -version

*Pastikan output menunjukkan versi Java 21.*

3

Buat Direktori & Download Installer

Siapkan folder khusus untuk server Anda:

bash
mkdir fabric-1.21.1 && cd fabric-1.21.1
wget https://maven.fabricmc.net/net/fabricmc/fabric-installer/1.0.1/fabric-installer-1.0.1.jar
4

Instalasi Fabric Server

Unduh data core server Minecraft:

bash
java -jar fabric-installer-1.0.1.jar server -mcversion 1.21.1 -downloadMinecraft

File fabric-server-launch.jar akan muncul setelah proses ini.

5

Konfigurasi EULA

Jalankan server sekali, lalu setujui EULA agar server bisa berjalan normal.

bash
java -Xmx8G -Xms4G -jar fabric-server-launch.jar nogui
nano eula.txt

Ubah eula=false menjadi eula=true. Simpan (CTRL+O, Enter) lalu keluar (CTRL+X).

6

Koneksi Internet (Playit.gg)

Instalasi agen tunneling playit.gg:

bash
curl -SsL https://playit-cloud.github.io/ppa/key.gpg | gpg --dearmor | sudo tee /etc/apt/trusted.gpg.d/playit.gpg >/dev/null
echo "deb [signed-by=/etc/apt/trusted.gpg.d/playit.gpg] https://playit-cloud.github.io/ppa/data ./" | sudo tee /etc/apt/sources.list.d/playit-cloud.list
sudo apt update
sudo apt install playit -y

Jalankan playit dan ikuti link untuk registrasi (Pilih Minecraft TCP, Port 25565).

7

Pasang Mod Performa

Sangat Disarankan! Unduh mod Lithium dan FerriteCore untuk mengurangi lag.

bash
cd mods
wget https://cdn.modrinth.com/data/P7dR8mSH/versions/bu0Zp0g1/fabric-api-0.102.0+1.21.1.jar
wget https://cdn.modrinth.com/data/ftdbN0KK/versions/9vH5C4iF/lithium-fabric-mc1.21.1-0.13.0.jar
wget https://cdn.modrinth.com/data/nmDcB62a/versions/7qX1Q7Xn/ferritecore-6.0.1-fabric.jar
cd ..

Konfigurasi Tambahan

Mode Offline

Untuk mengizinkan pemain dengan launcher non-premium (crack) bergabung.

Edit file server.properties:

online-mode=false

Menjalankan Server

Gunakan perintah ini setiap kali ingin menyalakan server dengan alokasi RAM optimal (8GB):

java -Xmx8G -Xms4G -jar fabric-server-launch.jar nogui

Panduan Sisi Client

Agar pemain bisa terhubung dengan lancar:

  • Gunakan Fabric Loader 1.21.1
  • Pasang Fabric API (Wajib)
  • (Opsional) Gunakan Iris Shaders & Sodium untuk visual & performa

Pemecahan Masalah

Server Gagal Start?

Pastikan Java yang terpasang adalah versi 21. Cek dengan java -version.

Tidak Bisa Join?

Pastikan tunnel playit sedang berjalan di terminal dan statusnya aktif.

Lag Berlebihan?

Pastikan mod performa (Lithium, FerriteCore) sudah ada di folder mods.
