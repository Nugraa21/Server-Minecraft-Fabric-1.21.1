Membangun Server Minecraft
Fabric 1.21.1

Panduan lengkap instalasi server profesional menggunakan GitHub Codespaces dan playit.gg.

Mulai Sekarang Pelajari Fitur

Fitur Utama

Performa Optimal

Menggunakan Fabric 1.21.1 yang sangat ringan dan efisien untuk server pribadi maupun komunitas.

Cloud Infrastructure

Konfigurasi siap pakai untuk GitHub Codespaces. Tidak perlu spesifikasi PC tinggi untuk menjalankan server.

Koneksi Aman

Integrasi playit.gg untuk tunneling aman tanpa perlu membuka port forwarding manual di router Anda.

Mod & Shader Ready

Mendukung penuh mod fabric-api, lithium, dan sodium untuk pengalaman bermain terbaik.

Panduan Instalasi Lengkap

Tahap 1: Persiapan Cloud
01

Akun GitHub & Fork

Daftar akun di GitHub. Setelah login, kunjungi repository ini dan klik tombol Fork di pojok kanan atas untuk menyalinnya ke akun Anda.

02

Setup Codespaces

Di repository hasil fork Anda, klik tombol hijau Code, pilih tab Codespaces, lalu klik Create codespace on main. Tunggu hingga terminal muncul.

Tahap 2: Konfigurasi Server
03

Install Java 21

Ketik perintah ini di terminal Codespaces untuk menginstal Java 21 (Wajib untuk 1.21.1).

sudo apt update && sudo apt install openjdk-21-jdk -y
04

Download & Build Fabric

Buat folder server dan unduh installer Fabric terbaru.

mkdir fabric-server && cd fabric-server
wget https://maven.fabricmc.net/net/fabricmc/fabric-installer/1.1.1/fabric-installer-1.1.1.jar
java -jar fabric-installer-1.1.1.jar server -mcversion 1.21.1 -downloadMinecraft
05

Setujui EULA

Ubah status eula menjadi true agar server bisa berjalan.

echo "eula=true" > eula.txt
Tahap 3: Online Tunneling
06

Install playit.gg

Agar server bisa diakses dari luar tanpa port forwarding.

curl -SsL https://playit-cloud.github.io/ppa/key.gpg | gpg --dearmor | sudo tee /etc/apt/trusted.gpg.d/playit.gpg >/dev/null
echo "deb [signed-by=/etc/apt/trusted.gpg.d/playit.gpg] https://playit-cloud.github.io/ppa/data ./" | sudo tee /etc/apt/sources.list.d/playit-cloud.list
sudo apt update && sudo apt install playit -y
playit
Tahap 4: Bergabung ke Server
07

Client Setup

Pastikan Anda menggunakan Fabric Loader 1.21.1 pada launcher Minecraft Anda. Masukkan alamat IP yang diberikan oleh playit.gg untuk terhubung.
