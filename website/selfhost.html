Persyaratan Hardware

Processor (CPU)

Minimal Dual Core 2.5GHz+. Disarankan Quad Core untuk performa tanpa lag.

RAM (Memory)

Minimal 4GB terdedikasi untuk server. Jika bermain di PC yang sama, gunakan 16GB total RAM PC.

Koneksi Internet

Upload speed minimal 10Mbps stabil jika ingin membuka server untuk publik via playit.gg.

Versi MC Perintah Instalasi (Universal) Installer
WINDOWS

Instalasi di Windows

Langkah mudah membuat server di PC/Laptop Windows 10/11.

1

Download & Install Java 21

Fabric 1.21.1 membutuhkan Java Development Kit (JDK) 21.

Download JDK 21 (Adoptium)

*Pilih installer .msi untuk Windows x64. Setelah download, install dan restart PC Anda.*

2

Persiapan Folder

Buat folder baru di Desktop atau Drive D: Anda. Beri nama MinecraftServer.

Jangan gunakan spasi pada nama folder agar tidak error.
3

Download Installer & Install

Download Universal Installer (.jar) atau (.exe). Letakkan di folder server.

Buka CMD (Command Prompt) di folder tersebut, lalu ketik perintah instalasinya (Contoh untuk 1.21.1):

CMD / Powershell
java -jar fabric-installer-1.0.1.jar server -mcversion 1.21.1 -downloadMinecraft

Fila fabric-server-launch.jar akan muncul.

4

Buat Run.bat

Buka Notepad, tempel kode di bawah, lalu Save As dengan nama Run.bat (Pilih "All Files" pada Save as type).

Run.bat
@echo off
java -Xms4G -Xmx6G -jar fabric-server-launch.jar nogui
pause
5

Jalankan Server

Double klik Run.bat. Console akan muncul dan menutup.
Buka eula.txt, ubah eula=false menjadi true.
Jalan Run.bat lagi. Selesai!

LINUX (Ubuntu/Debian)

Instalasi di Linux

Cocok untuk penggunaan VPS atau OS Linux Desktop.

1

Install Java 21

Gunakan terminal untuk menginstal OpenJDK 21.

bash
sudo apt update
sudo apt install openjdk-21-jdk -y
java -version
2

Setup Folder & Download Installer

bash
mkdir fabric-server
cd fabric-server
wget https://maven.fabricmc.net/net/fabricmc/fabric-installer/1.0.1/fabric-installer-1.0.1.jar
3

Install Server Core

bash
java -jar fabric-installer-1.0.1.jar server -mcversion 1.21.1 -downloadMinecraft
4

Start Script

Buat file start.sh:

bash
echo "java -Xms4G -Xmx6G -jar fabric-server-launch.jar nogui" > start.sh
chmod +x start.sh

Jalankan sekali dengan ./start.sh, lalu edit eula.txt dan setujui. Jalankan lagi ./start.sh.

Akses Publik (Playit.gg)

Agar teman Anda bisa bergabung ke server Localhost Anda.

playit.gg (Mudah)

Cara termudah tanpa setting router modem.

  1. Download aplikasi playit.gg.
  2. Jalankan aplikasinya (Double click di Windows, atau via terminal di Linux).
  3. Buka link 'Claim' yang muncul.
  4. Tambahkan tunnel Minecraft Java (Local Port: 25565).
  5. Selesai! Gunakan domain playit (contoh: nugra.playit.gg) untuk join.

Port Forwarding (Manual)

Metode traditional yang memerlukan akses admin router.

1. Login router (192.168.1.1/192.168.100.1).
2. Menu Network / Forwarding.
3. Buka Port 25565 TCP/UDP.
4. Arahkan ke IP Local PC Anda (Cek via ipconfig, misal 192.168.1.10).

Masalah Lokal Umum

Windows Firewall?

Jika teman tidak bisa join meski sudah playit.gg, izinkan "Java Platform SE Binary" di Windows Firewall (Private & Public).

Run.bat langsung menutup?

Tambahkan baris pause di baris terakhir file Run.bat untuk melihat pesan errornya (biasanya karena Java belum terinstall).

RAM Penuh?

Jangan alokasikan RAM melebihi kapasitas PC. Jika PC punya 8GB, server max 4GB saja agar Windows tidak crash.
