Instalasi Mods

Cara menambahkan modifikasi gameplay atau performa ke server Anda.

1

Download Mod yang Kompatibel

Pastikan Anda mendownload mod khusus untuk Fabric 1.21.1. Situs terpercaya:

  • Modrinth (Recommended)
  • CurseForge
Wajib: Hampir semua mod Fabric membutuhkan Fabric API. Pastikan file fabric-api-xxxx.jar selalu ada di folder mods.
2

Upload ke Server

Pindahkan file .jar mod yang sudah didownload ke dalam folder mods di direktori server Anda.

Struktur Folder
server-folder/
├── mods/
│   ├── fabric-api-xxxx.jar  <-- WAJIB
│   ├── lithium-mc1.21.1.jar
│   └── mod-pilihan-anda.jar
└── ...
3

Restart Server

Mods baru hanya akan terbaca setelah server direstart (Stop lalu Start kembali). Cek console/terminal untuk memastikan tidak ada error crash.

Pasang Datapacks

Menambah fitur custom tanpa harus install mod di sisi client pemain.

Langkah Pemasangan

1. Siapkan file datapack (biasanya .zip) atau folder datapacknya.

2. Buka folder world server Anda, lalu masuk ke folder datapacks.

server-folder/world/datapacks/

3. Masukkan file zip datapack ke situ.

4. Jika server sedang menyala, ketik perintah /reload di console atau di game (jika OP).

5. Cek apakah datapack aktif dengan perintah /datapack list.

Ganti World / Peta

Cara menggunakan world custom atau map parkour/adventure pilihan Anda.

A

Metode Timpa (Replace)

1. Matikan server.

2. Hapus folder world yang ada di server (atau rename jadi world_backup).

3. Copy folder world custom Anda ke folder server.

4. Rename folder tersebut menjadi world agar sesuai dengan setting default.

5. Nyalakan server.

B

Metode Config

Jika Anda tidak ingin mengubah nama folder map Anda:

1. Copy folder map (misal: map_parkour) ke folder server.

2. Edit file server.properties.

3. Cari bagian level-name=world dan ubah menjadi level-name=map_parkour.

4. Simpan dan restart server.

Konfigurasi Pemain & Izin

Mengatur siapa yang bisa masuk dan siapa yang menjadi admin.

Izinkan Player Crack

Secara default, server hanya mengizinkan akun original (Premium). Untuk mengizinkan user TLauncher/Crack:

  1. Matikan server.
  2. Buka file server.properties.
  3. Cari online-mode=true.
  4. Ubah menjadi online-mode=false.
  5. Simpan dan nyalakan server.
Peringatan: Skin pemain mungkin tidak muncul, dan keamanan akun sedikit berkurang.

Memberikan Izin OP (Admin)

Operator (OP) memiliki akses ke semua command seperti /gamemode, /ban, dll.

Ketik di Terminal Server (Tanpa /)
op NamaPlayer

Untuk mencabut izin OP:

deop NamaPlayer

Whitelist (Mode Privat)

Agar hanya orang tertentu yang bisa masuk:

Di Terminal
whitelist on
whitelist add NamaPlayer

Pastikan white-list=true di server.properties jika command tidak bekerja.
