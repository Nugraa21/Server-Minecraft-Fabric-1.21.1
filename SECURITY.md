# Kebijakan Keamanan

Kami sangat serius dalam menjaga keamanan konfigurasi server dan data pengguna. Panduan ini dimaksudkan untuk membantu Anda menjalankan server dengan aman.

## Melaporkan Masalah Keamanan

Jika Anda menemukan celah keamanan dalam konfigurasi yang kami sediakan:
- **Tolong jangan melaporkannya melalui GitHub Issues publik.**
- Silakan hubungi pengelola secara langsung melalui profil GitHub atau kirimkan laporan melalui mekanisme kontak yang tersedia.
- Kami akan berusaha merespons laporan Anda dalam waktu 48 jam dan bekerja sama untuk memperbaiki masalah tersebut sebelum dipublikasikan.

## Praktik Keamanan yang Disarankan
- **Password**: Jangan pernah membagikan password database atau token `playit.gg` Anda.
- **Whitelist**: Selalu aktifkan fitur `whitelist` di server Anda untuk mencegah akses tidak dikenal.
- **Sistem Update**: Pastikan Java dan sistem operasi (dalam Codespaces) selalu diperbarui ke versi terbaru.
